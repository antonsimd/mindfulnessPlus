# iOS Mindfulness Journaling App

## Overview
This is an iOS-based mindfulness journaling app currently under development. The app aims to provide users with a platform to record their thoughts, feelings, and experiences in a mindful way.

Made to grasp an understanding of MVVM architecture and iOS development.

## Prerequisites
- Xcode 12.0 or higher
- Swift 5.0 or higher

## Setup
1. Clone this repository to your local machine.
2. Open the `.xcodeproj` file in Xcode.
3. Install the required dependencies via CocoaPods with `pod install`.

## Usage
1. Build and run the project in Xcode.
2. Install the app on your iOS device or simulator.

## App Features (Under Development)
- **Journal Entries**: Users can create, edit, and delete journal entries.
- **Mindfulness Prompts**: The app provides prompts to guide users in their journaling.
- **Reminders**: Users can set reminders to journal at specific times.
